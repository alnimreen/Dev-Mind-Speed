# Setup Instructions
## Prerequisites
- Java Development Kit (JDK) 17 
- Maven
- MySQL 

## Build the Project
mvn clean install

## Run the Application
mvn spring-boot:run

## Access the Application
The application will be running on http://localhost:8080.
